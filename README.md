# truffle-react-template

> A pure truffle-react template to use for web3 React developers!

### Run the template:

- Run ganache UI (CLI)
- Run truffle
```linux
truffle compile
truffle migrate --network develop
```
- Run react
```linux
cd client
npm install
npm run start
```
